## Prerequisites

- Python 3.10.6
- `venv` module

## Installation

1. **Create a virtual environment:**

  ```bash
  python3 -m venv venv
  ```

2. **Activate the virtual environment:**

  - On macOS and Linux:

    ```bash
    source venv/bin/activate
    ```

  - On Windows:

    ```bash
    .\venv\Scripts\activate
    ```

3. **Install the required packages:**

  ```bash
  pip install -r requirements.txt
  ```

## Running the Project

1. **Run the main Python script:**

  ```bash
  python main.py
  ```
